단기과열종목 계산기를 만들어서 소개하는 글을 써보고자 합니다.

## 왜 만들었나

저도 주식 투자를 하는데, 단기과열종목 지정 조건을 확인하는 게 너무 귀찮았습니다.  
단기과열종목에 걸리면 3일간 단일가거래에 걸리기 때문에 신경이 쓰이더군요.

그런데 단기과열종목 계산을 위해서는 지난 40영업일 동안의 종가와 거래량 데이터가 필요합니다.  
또한, 주가상승률, 거래회전율, 주가변동성 등을 계산해야 하는데 개인투자자가 이걸 일일이 하기는 너무 어렵습니다...

그리고 네이버 종목토론방 보면 항상 "내일 단일가다" "아니다" 이런 얘기들이 많더라구요.  
이게 그렇게 어려운 건 아닌데 (돈이 안 돼서 그렇겠죠..?) 쉽게 계산해주는 서비스 하나 정도는 있으면 좋지 않을까 해서 직접 만들었습니다.

## 어떤 서비스인가

단기과열종목 계산기라고 이름 지었습니다.

**사용법은 간단합니다:**

1. 종목 코드 입력  
2. 검색 버튼 클릭  
3. 결과 확인  
4. 끝입니다.

종목코드를 검색만 하면 아래처럼 결과를 계산해서 보여드립니다.

![단기과열종목 계산기 결과 화면 스크린샷](/introduction.png)

단기과열종목 지정 조건인 주가상승률, 거래회전율, 주가변동성 이 3가지 조건을 자동으로 계산해줍니다.  
40일치 데이터 수집부터 복잡한 공식 적용까지 한 번에 가능합니다.



## 주의사항

- 가격 괴리율로 인한 단기과열 예고는 계산 안 됩니다.  
- 실제 단기과열 예고는 공시로 확인해야 합니다.  
- 그냥 참고용입니다. 투자 권유 아님  

단기과열 지정되면 3일간 단일가거래 걸리는 거 다들 아시죠?  
그래서 미리 체크해보는 용도로 쓰시면 될 것 같습니다.
